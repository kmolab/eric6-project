UserInterface.processArgs

processArgs(args)

Public method to process the command line args passed to the UI.

args
list of files to open
The args are processed one at a time. All arguments after a '--' option are considered debug arguments to the program for the debugger. All files named before the '--' option are opened in a text editor, unless the argument ends in .e4p, then it is opened as a project file. If it ends in .e4m or .e5m, it is opened as a multiproject.
