eric6.WebBrowser.Network

Package containing network related modules.

Modules

EricSchemeHandler Module implementing a scheme handler for the eric: scheme.
NetworkManager Module implementing a network manager class.
NetworkUrlInterceptor Module implementing a class to handle URL requests before they get processed by QtWebEngine.
QtHelpSchemeHandler Module implementing a scheme access handler for QtHelp.
SendRefererWhitelistDialog Module implementing a dialog to manage the Send Referer whitelist.
SslErrorExceptionsDialog Module implementing a dialog to edit the SSL error exceptions.
UrlInterceptor Module implementing an URL interceptor base class.
